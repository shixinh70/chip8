# Chip-8 Emulator (Interpreter) build on C and SDL2
## Dependencies
* gcc 
* make
* SDL2 (libsdl2, libsdl2-dev)

```
sudo apt-get install gcc make libsdl2-2.0-0 libsdl2-dev
```

## Build
* make

## Run
* ./chip8 ./games/<game_names>
